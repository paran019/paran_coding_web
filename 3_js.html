1.JavaScript

1)특징
- body 태그 안에 존재
- 자극이나 조건에 반응하는 유동적인 언어
2)사용 방법
- scrip 태그
- input 태그와 event
- console : 웹브라우저 개발자 도구에서 사용
사용중 실행을 미루고 싶을 땐 shift + enter

2.Data types : String(문자열) & Number(숫자)

1)String
- 큰 따옴표 or 작은 따옴표
ex.) 텍스트의 글자 수를 알고 싶을 때
"hello world" .length

2)Number
- 산술연산자 : + - * /
- 대입연산자 : =
ex.) 1+1의 값을 알림 창으로 띄우고 싶을 때
alert(1+1);

3)문자열과 숫자를 정확히 표기해야 하는 이유.
"1"+"1" = "11"
1+1 = 2

4)변수를 쓸 땐 앞에 var(variable) 붙이기
